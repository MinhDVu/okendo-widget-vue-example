# widget-vue-example

## Project setup
Clone the okendo-widgets-vue project

```
npm install
npm link @okendo/okendo-widgets-vue
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
